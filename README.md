## L’s Buff Bars
A World of Warcraft addon that provides customizable buff‑bars.

## Installation
1. Download or clone the repository.
2. Place the addon folder into:  
   `World of Warcraft/_retail_/Interface/AddOns/`
3. Launch the game and enable **L’s Buff Bars** in the AddOns menu.
